<h1 align="center">SCUM Server Manager</h1>

<p align="center">
  A powerful tool for launching, monitoring, automating, and managing a dedicated SCUM server.

  Designed for server owners who want an easier way to control their server, manage players, automate maintenance tasks, and integrate Discord administration tools.
</p>

![SCUM](https://img.shields.io/badge/Game-SCUM-red)

---

# ✨ Features

## 🔧 Server Management

- Start / Stop / Restart the server
- Automatic server updates
- Scheduled automatic restarts
- Install SteamCMD directly from the application
- Install and update the SCUM dedicated server
- Support for custom server paths
- Real-time server monitoring
- Live server console and log viewer

---

## 👥 Player Management

- Online players list
- Player connection history
- Ban / Unban players
- Automatic country-based bans
- Assign administrator permissions via SteamID
- Player activity monitoring
- Warning system

---

## 📊 Monitoring & Automation

- Real-time action monitoring
- Vehicle activity tracking
- Player activity logging
- Automatic maintenance tools
- Integrated log viewer
- Server event tracking

---

# 🤖 Discord Integration

Control and monitor your SCUM server directly from Discord.

## Discord Features

### 🖥 Server Controls

- Start server
- Restart server
- Stop server
- View server status

### 👥 Player Controls

- View all players
- View online players
- View banned players
- Ban / Unban players
- Send player warnings

### 📡 Status & Monitoring

- Live server status
- Server activity in bot presence
- Chat message monitoring
- Player event notifications
